MSR Tier Tagger — Feature Roadmap
Overview

MSR Tier Tagger is a competitive ranking and visualization mod designed to display player skill tiers, track performance across multiple PvP gamemodes, and automate ranking updates through structured match results.
Version 1 — Core Tier Display
Goal: Establish the foundation for player ranking visibility.

Features
Rank Display System
Displays player MSR tier next to their name in chat and nametags
Highlights highest achieved tier
Client-Side Visibility
Tier tags are only visible to users with the mod installed
Gamemode Integration
Custom icons for each supported gamemode
Tier Structure
Tier 1: Gold
Tier 2: Silver
Tier 3: Bronze
Distribution Model
Basic Version: Current Tierlist
Premium Version: Premium Only Tierlist
Version 2 — Data & Interaction Expansion
Goal: Introduce player data access, interactivity, and recognition systems.

Features
Command System
/msr profile <IGN> — View player stats and tier
/msr leaderboards overall — Global rankings
/msr leaderboards <gamemode> — Gamemode-specific rankings
Data Integration
Full MSR tier list database embedded into the mod
Real-time access to rankings and player stats
Custom Recognition Tags
Unique nametags and colors for:
Tournament Winners
King of PvP
Developers
Version 3 — Automated Ranking System
Goal: Eliminate manual ranking by introducing automated match evaluation.

Features
Automatic Match Detection
Detects duels between MSR-ranked players
Validates correct gamemode and match format
Supported Competitive Formats
Sword — FT10
Axe — FT10
UHC — FT7
Mace — FT5
Crystal — FT5
Hybrid — FT3
SMP — FT3
DSMP — FT3
DPot — FT5
NPot — FT5
Automated Result Processing
Tracks match outcomes
Applies promotions and demotions based on a predefined ruleset
Dynamic System Updates
Automatic tier adjustments
Real-time leaderboard updates
Future Considerations (Optional Expansion)
Web dashboard for stats and rankings
Seasonal ranking resets and rewards
Anti-abuse / match validation safeguards
API for server integration
Replay or match history tracking
